Tiki Toki Games No Wifi

Project Overview

"Tiki Toki Games No Wifi" is a mobile game compilation designed to capture the fun of viral TikTok filter games in a single, accessible, and offline-playable package. Recognizing the popularity of these interactive trends, I set out to create a collection that allows users to enjoy their favorite challenges anytime, without needing an internet connection. The project was executed under a strict self-imposed deadline to simulate a fast-paced production environment and sharpen my rapid prototyping skills.

Challenges & Solutions

The core challenge of this project was the demanding production schedule: to conceptualize, design, create assets for, and program a new, fully functional minigame from scratch every single week. I successfully met this aggressive goal by implementing a highly disciplined and structured weekly routine.

Tools & Technologies

This game was developed using a Unity as a game engine and Blender for asset creation.
